# SellSync 사용자 온보딩 개선 사항

## 📋 개요

사용자 가이드(SellSync_사용자_가이드_v2.pdf)를 기반으로 첫 가입 사용자가 시스템에 쉽게 적응할 수 있도록 다음과 같은 기능들을 구현했습니다.

## 🎯 구현된 기능

### 1. 빠른 시작 가이드 위젯
**파일**: `components/help/QuickStartGuide.tsx`

- 대시보드에 표시되는 체크리스트 형식의 가이드
- 5단계 설정 프로세스 시각화
- 진행률 표시 및 완료 상태 추적
- 각 단계별 바로가기 링크 제공

**표시 단계**:
1. ERP 연동 설정
2. 쇼핑몰 연동
3. ERP 품목 동기화
4. 상품 매핑
5. 주문 동기화

### 2. 도움말 패널
**파일**: `components/help/HelpPanel.tsx`

- 헤더의 물음표 아이콘 클릭 시 표시
- 검색 가능한 FAQ
- 카테고리별 질문 필터링
- 빠른 링크 (사용자 가이드, 문의하기)

**포함된 FAQ 카테고리**:
- API 연동
- ERP 연동
- 스토어 관리
- 전표 생성
- 동기화

### 3. 컨텍스트 도움말
**파일**: `components/help/ContextHelp.tsx`

- 폼 필드 옆의 물음표 아이콘
- 호버 시 툴팁으로 설명 표시
- 온보딩 단계의 모든 입력 필드에 적용

### 4. FAQ 페이지
**경로**: `/help/faq`
**파일**: `app/(dashboard)/help/faq/page.tsx`

- 전체 FAQ 목록
- 검색 및 카테고리 필터
- 외부 링크 (네이버 커머스 API 센터, 쿠팡 Wing 등)
- 15개의 자주 묻는 질문 포함

### 5. 사용자 가이드 페이지
**경로**: `/help/guide`
**파일**: `app/(dashboard)/help/guide/page.tsx`

- 탭 형식의 구조화된 가이드
- 4개 섹션:
  - 개요
  - 연동 설정 (ERP, 네이버, 쿠팡)
  - 주문 동기화
  - 상품 매핑
- 단계별 상세 설명
- 외부 링크 및 주의사항

### 6. 개선된 온보딩 단계

#### ERP 연동 단계
**파일**: `components/onboarding/steps/ErpSetupStep.tsx`

추가된 내용:
- API 인증키 발급 방법 안내
- 이카운트 ERP 바로가기 링크
- 각 입력 필드에 컨텍스트 도움말
- 플레이스홀더 예시

#### 스토어 연동 단계
**파일**: `components/onboarding/steps/StoreSetupStep.tsx`

추가된 내용:
- 마켓플레이스별 API 발급 방법 (네이버/쿠팡)
- 외부 링크 (네이버 커머스 API 센터, 쿠팡 Wing)
- 주의사항 표시 (유효기간, 권한 등)
- 각 입력 필드에 컨텍스트 도움말

### 7. 사이드바 개선
**파일**: `components/layout/sidebar.tsx`

- 하단에 "도움말" 섹션 추가
- 사용자 가이드 링크
- FAQ 링크

### 8. 헤더 개선
**파일**: `components/layout/header.tsx`

- 도움말 버튼 추가 (물음표 아이콘)
- 클릭 시 도움말 패널 표시

## 📝 주요 개선 사항

### 1. 정보 접근성 향상
- 필요한 정보를 바로 확인할 수 있는 컨텍스트 도움말
- 외부 사이트로 바로 이동할 수 있는 링크
- 검색 가능한 FAQ

### 2. 시각적 가이드
- 진행률 표시
- 단계별 체크리스트
- 컬러 코딩 (성공/경고/오류)

### 3. 실용적인 정보 제공
- 실제 API 발급 절차
- 플레이스홀더 예시
- 주의사항 및 팁

### 4. 사용자 경험 개선
- 원클릭으로 접근 가능한 도움말
- 상황에 맞는 도움말 표시
- 깔끔한 UI/UX

## 🚀 사용 방법

### 1. 빠른 시작 가이드 사용
- 대시보드 접속 시 자동으로 표시됨
- 각 단계 클릭하면 해당 페이지로 이동
- 모든 단계 완료 시 자동으로 숨김

### 2. 도움말 패널 사용
- 헤더 우측의 물음표 아이콘 클릭
- 질문 검색 또는 카테고리 선택
- 관련 답변 확인

### 3. FAQ 페이지 사용
- 사이드바 하단 "자주 묻는 질문" 클릭
- 검색 또는 카테고리 필터 사용
- 15개의 주요 질문 확인

### 4. 사용자 가이드 사용
- 사이드바 하단 "사용자 가이드" 클릭
- 탭을 통해 원하는 섹션 선택
- 단계별 상세 설명 확인

## 📊 포함된 정보

### API 연동 가이드
- **네이버 스마트스토어**: Client ID/Secret 발급 방법
- **쿠팡**: Vendor ID, Access Key, Secret Key 발급 방법
- **Ecount ERP**: API 인증키 발급 방법

### 자주 묻는 질문 (15개)
1. 네이버 커머스 API 키 확인 방법
2. 쿠팡 API 키 만료 시 대응
3. ERP API 인증키 발급
4. 거래처/창고 코드 확인
5. 스토어 비활성 문제
6. 여러 스토어 동시 연동
7. 주문 동기화 주기
8. 전표 생성 실패
9. 전표 생성 실패 처리
10. 상품 매핑 개념
11. AI 추천 매핑 정확도
12. 동기화 실패
13. ERP 품목 동기화 시점
14. 송장 자동 반영
15. 문제 발생 시 연락처

## 🔄 향후 개선 가능 사항

1. **비디오 튜토리얼**: 각 단계별 동영상 가이드
2. **인터랙티브 투어**: 첫 로그인 시 단계별 안내
3. **챗봇 지원**: 실시간 질문 응답
4. **다국어 지원**: 영어 등 추가 언어
5. **사용 패턴 분석**: 어려워하는 단계 파악
